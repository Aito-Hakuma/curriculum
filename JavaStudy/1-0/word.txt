1. OS
Operation Systemの略で、アプリやデバイスを動作させるための基本となるソフトウェアのこと。

2. Windows
マイクロソフト社が開発したOS。

3. macOS(Mac OS X)
Apple社が開発したOS。

4. UNIX
現存する中で最も古いOSの1つ。
アメリカ最大手の電話会社であるAT&T者が1968年に開発したOS。
UNIXを利用するためには、ライセンスの取得が必要である。

5. Linux
リーナス・トーバルズによって開発されたOS。
UNIXと混合されがちであるが、Linuxはオープンソースソフトウェアとして広く公開・配布されているため、誰でも無料で使用することができる。

6. HTML
HyperText Markup Languageの略。
ウェブページを作成するために開発された言語であり、見出しやヘッダ情報などWebページの文書構造を形作るための言語。

7. CSS
Cascading Style Sheetsの略。
文書構造にデザインを施し、見栄えを整える役割を担っている言語。

8. MySQL
世界で最も利用されているデータベース管理システム。

9. PHP
動的にWebページを生成することができるサーバーサイドのスクリプト言語。

10. JavaScript
ユーザ側のWebブラウザとWebサイトまたはウェブサービスの相互間のやり取りを、円滑にするために使用されるプログラミング言語。

11. jQuery
JavaScriptでできることを、より簡単な記法で実現できるように設計されたJavaScriptライブラリ。

12. Bootstrap
HTML/CSS/JavaScriptから構成される最も有名なWEBフレームワーク。

13. SE(システムエンジニア)
クライアントにヒアリングを行い、要求に最適な仕様のシステム開発を行う役割のこと。

14. PG(プログラマー)
コンピューターを動かす「プログラム言語」を用いて、さまざまなシステムやソフトウェアを作る仕事のこと。

15. PM(プロジェクトマネージャー)
プロジェクト全体の進行を管理し、予算や品質、納期、成果物のクオリティに対して全責任を持つ役職のこと。

16. Webデザイナー
Webページのデザインや企画設計、コーディングを行う役割のこと。

17. ディレクトリ
フォルダのこと。

18. ルート・ディレクトリ
階層型ファイル構造における最上階層のディレクトリのこと。

19. カレント・ディレクトリ
ユーザーが現時点で作業を行っているディレクトリのこと。

20. HTTP
Hyper Text Transfer Protocolの略。
WebサーバとWebクライアントの間でデータの送受信を行うために用いられるプロトコル。

21. HTTPS
Hyper Text Transfer Protocol Secureの略。
HTTPとほとんど同様であるが、HTTPSでは通信内容が暗号化されている。

22. SSL
Secure Sockets Layerの略。
インターネット上におけるウェブブラウザとウェブサーバ間でのデータの通信を暗号化し、送受信させる仕組み。

23. IPアドレス
Internet Protocol Addressの略。
パソコンやスマートフォンなどネットワーク上のすべての機器を識別するために割り当てられている、インターネット上での住所のような役割。

24. サーバー
ホームページを表示するために必要となる情報を格納しておく場所のこと。

25. ドメイン
Webサイトがどこにあるかを判別する情報。

26. キャッシュ
訪問したウェブページの情報を一時的に保存する仕組み。

27. 単体テスト
プログラムを構成する比較的小さな単位が個々の機能を正しく果たしているかどうかを検証するテスト。

28. 結合テスト
システム開発におけるプログラムの検証作業の中でも、手続きや関数といった個々の機能を結合させて、うまく連携・動作しているかを確認するテストのこと。

29. Git
プログラムソースなどの変更履歴を管理する分散型のバージョン管理システムのこと。
